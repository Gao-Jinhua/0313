🌍 什么是资源与环境学？

资源与环境学是研究自然资源的形成、分布、利用及其与环境之间相互关系的综合性学科。主要涵盖：

  • 土地资源：研究土壤的形成演变、性质特征及合理利用
  • 水资源：研究水文循环、水资源评价与可持续利用
  • 生物资源：研究生物多样性保护与生态系统服务
  • 环境保护：研究污染防治、生态修复与环境管理

💡 关键认知：资源的合理利用与环境的有效保护是实现可持续发展的核心。

🧪 土壤学基础笔记

土壤是由矿物质、有机质、水分、空气和微生物等组成的复杂体系。

土壤组分 比例（体积） 功能
矿物质 ~45% 提供矿质养分，构成土壤骨架
有机质 ~5% 提供养分，改善土壤结构
水分 ~25% 溶解和传输养分
空气 ~25% 为根系和微生物提供氧气

土壤质地三角形：根据砂粒、粉粒、黏粒的比例，将土壤分为砂土、壤土、黏土等类型。

📊 R 语言学习笔记

R 是一种用于统计计算和图形的编程语言，在环境科学数据分析中被广泛使用。

# 基本数据操作示例
library(tidyverse)

# 读取数据
soil_data <- read_csv("soil_samples.csv")

# 数据摘要
summary(soil_data)

# 可视化：pH 值分布
ggplot(soil_data, aes(x = pH)) +
  geom_histogram(fill = "#4CAF50", color = "white", bins = 20) +
  labs(
    title = "土壤 pH 值分布",
    x = "pH 值",
    y = "频次"
  ) +
  theme_minimal()
